# Habit Tracker 📝🔥


A simple terminal-based **habit tracker** built in Python.  
Keep track of your daily habits, complete them, and maintain streaks!  

---

## **Badges**

![Python](https://img.shields.io/badge/Python-3.12-blue?logo=python)
![GitHub](https://img.shields.io/badge/GitHub-repo-black?logo=github)

---

## **Features**

- Add, complete, and delete habits  
- Track streaks for daily consistency 🔥  
- Daily reset for unfinished habits  
- Colorful terminal interface using **Colorama**  
- Modular Python code (`main.py`, `habits.py`, `storage.py`, `ui.py`)
